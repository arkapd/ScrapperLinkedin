### 🎨 Visual Language

#### Colors
- `--primary-bg`: `#0a0e14` (Deep Space Dark)
- `--card-bg`: `rgba(255, 255, 255, 0.05)` (Glass Base)
- `--accent-blue`: `#00a8ff`
- `--text-main`: `#ffffff`
- `--text-muted`: `rgba(255, 255, 255, 0.7)`

#### Glassmorphism Specs
- `backdrop-filter: blur(10px)`
- `border: 1px solid rgba(255, 255, 255, 0.1)`
- `box-shadow: 0 8px 32px 0 rgba(0, 0, 0, 0.37)`

#### Breakpoints
- **Mobile**: `up to 768px` (Single column cards)
- **Desktop**: `769px and above` (Grid layout)
