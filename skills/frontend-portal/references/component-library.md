### 🧱 UI Components

#### 1. The Job Card
- **Logic**: Generated by `renderJobs(data)` in `script.js`.
- **Fields**: 
    - `role`: Upper case transformation.
    - `location`: Map icon + text.
    - `snippet`: Truncated at 150 chars.
    - `contact`: Actionable links (mailto/tel).

#### 2. Filter Bar
- **Dropdowns**: Populated from unique values in `jobs.json`.
- **Auto-Sync**: Filters combined using `AND` logic.

#### 3. Last Updated HUD
- **Location**: Bottom right or Header subtitle.
- **Logic**: Shows time elapsed since last successful polling.
